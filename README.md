# Dashboard Executivo de Vendas e Distribuição

## Objetivo

Projeto desenvolvido em Power BI para análise de vendas, participação de mercado, distribuição geográfica e concentração de produtos.

O objetivo é fornecer uma visão executiva do desempenho comercial, permitindo identificar padrões de vendas, participação das redes varejistas, distribuição regional e oportunidades de negócio.

## Competências Demonstradas

- Modelagem Dimensional (Star Schema)
- Power BI
- Power Query
- DAX
- Storytelling Analítico
- Visualização de Dados
- Análise Exploratória de Dados

## Tecnologias Utilizadas

- Power BI Desktop
- Power Query (M)
- DAX
- Microsoft Excel

## Indicadores Principais

- Total de Vendas
- Quantidade de Produtos
- Quantidade de Lojas
- Quantidade de Cidades

## Análises Disponíveis

### Pareto de Produtos
Identificação da concentração das vendas por SKU e avaliação da dependência comercial de produtos específicos.

### Participação por Rede
Comparação da participação de cada rede varejista no volume total de vendas.

### Distribuição Geográfica
Análise da distribuição das vendas por Unidade Federativa (UF).

### Ranking de Lojas
Identificação das lojas com melhor desempenho.

### Participação dos Produtos por Rede
Avaliação da representatividade de cada produto dentro das redes varejistas.

## Principais Insights

- Baixa concentração das vendas entre os produtos.
- Portfólio equilibrado entre os SKUs analisados.
- Participação relativamente homogênea entre as redes varejistas.
- Distribuição geográfica pulverizada entre os estados avaliados.
